## [1.0.0] — 2026-02-26

### ✨ الميزات الجديدة

- **شاشة تسجيل الدخول** — واجهة ترحيب مع دعم OAuth عبر GitHub ووضع الضيف
- **شاشة المستودعات** — تصفح المستودعات مع بحث وتصفية (عام/خاص)
- **محرر الكود** — محرر Monaco مع دعم أكثر من 10 لغات برمجة
- **مستعرض الملفات** — شجرة ملفات تفاعلية مع أيقونات حسب نوع الملف
- **تبويبات متعددة** — فتح عدة ملفات والتنقل بينها
- **مساعد AI** — دردشة مدمجة للمساعدة في كتابة وشرح الكود
- **تكامل GitHub** — تصفح المستودعات، تحرير الملفات، Commit & Push
- **لوحة Git** — عرض حالة المستودع، الفروع، وسجل الالتزامات
- **إدارة الفروع** — إنشاء، حذف، والتبديل بين الفروع
- **معاينة مباشرة** — عرض HTML, CSS, JavaScript, Markdown, JSON
- **شاشة الإعدادات** — تخصيص حجم الخط، Tab Size، التفاف الأسطر، الخريطة المصغرة
- **شريط الحالة** — عرض اللغة، عدد الأسطر، والملف النشط
- **حوار الالتزام** — نافذة مخصصة لكتابة رسالة الالتزام
- **سمة سينمائية داكنة** — تصميم احترافي مع تأثير grain overlay وألوان دافئة

### 🏗️ البنية التحتية

- إعداد Vite + React 18 + TypeScript
- تكامل Tailwind CSS مع نظام تصميم مخصص (Design Tokens)
- إعداد Vitest مع React Testing Library
- إعداد Edge Functions للتكامل مع GitHub API
- تكوين ESLint للجودة

### 🧪 الاختبارات

- اختبارات وحدة لـ `editorStore` (flattenFiles, getFileIcon)
- اختبارات وحدة لإعدادات المحرر (DEFAULT_EDITOR_SETTINGS)
- اختبارات واجهة لشاشة تسجيل الدخول (LoginScreen)
- اختبارات واجهة لشاشة الإعدادات (SettingsScreen)
- اختبارات واجهة لحوار الالتزام (CommitDialog)
- اختبارات تكامل للمكونات (FileExplorer, TabBar, StatusBar)
- اختبارات تكامل للتنقل بين الشاشات (Navigation)
- **76 اختبار** — نسبة النجاح 100%
